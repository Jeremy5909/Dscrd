## DSCRD
dscrd is a Discord CLI
## Hosting locally
### Clone and cd into the repository
`git clone https://github.com/Jeremy5909/Dscrd.git && cd Dscrd`
### Install the neccessary packages
`npm i`
### Setup dotenv secrets
`touch .env`
Fill this in
```
ClientID=..
ClientSecret=..
PORT=8080
token=..
```
### Start listening
`node .`
